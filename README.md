# MPA Site Bible

Interactive operational command interface for Asian Games 2026 venue management and overlay systems.

## Stack

- Next.js
- TypeScript
- TailwindCSS

## Venue Systems

The application presents a command-center view of core venue systems, operational readiness, risk exposure, timeline milestones, and schematic site zones.

## Overlay

Temporary infrastructure, field-of-play support, compound planning, site furniture, and delivery interfaces for Games-time operations.

## Security

PSA, VSA, ACP, accreditation, secure perimeter, screening, and access-control logic for protected venue areas.

## Power

Temporary generation, fuel logistics, cable containment, distribution, redundancy, and commissioning dependencies.

## Broadcast

Broadcast compound, HB/RHB support, camera/audio interfaces, media operations, cable routes, and late-change controls.

## Logistics

Delivery routes, service access, cold storage, waste operations, vehicle movement, traffic control, and material flow.

## HSE

Medical support, emergency response, heat/weather exposure, fire inspection, site safety, and operational risk controls.

## BOQ Intelligence

Future intelligence layer for quantities, scope ownership, procurement packages, system dependencies, and cost-risk visibility.

## Run Commands

```bash
npm install
npm run dev
npm run build
```
